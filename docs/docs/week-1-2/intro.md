# Introduction to Physical AI & Humanoid Robotics

Welcome to the Physical AI & Humanoid Robotics Textbook. This 13-week curriculum is designed for industry engineers with Python knowledge who want to learn about Physical AI and humanoid robotics concepts and implementation techniques.

## Learning Objectives

By the end of this textbook, you will be able to:
- Understand the foundations of Physical AI and embodied intelligence
- Work with ROS 2 for robotics applications
- Use Gazebo for robot simulation
- Implement AI-powered perception and manipulation using NVIDIA Isaac
- Develop humanoid robot kinematics and dynamics
- Create conversational AI for robots
- Build an autonomous humanoid pipeline integrating speech, planning, navigation, perception, and manipulation

## Course Structure

This textbook is organized into 13 weeks:
- **Weeks 1-2**: Introduction to Physical AI
- **Weeks 3-5**: ROS 2 Fundamentals
- **Weeks 6-7**: Robot Simulation with Gazebo
- **Weeks 8-10**: NVIDIA Isaac Platform
- **Weeks 11-12**: Humanoid Robot Development
- **Week 13**: Conversational Robotics
- **Capstone**: Build an autonomous humanoid pipeline

## Prerequisites

- Python programming knowledge
- Basic understanding of robotics concepts (helpful but not required)
- Familiarity with command line tools
